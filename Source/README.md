# Futoshiki Solver — CSC14003

## Cài đặt
pip install -r requirements.txt

## Cách chạy
python main.py <input_file> <algorithm> [output_file]

## Algorithms
- bt    : Backtracking
- fc    : Forward Chaining
- bc    : Backward Chaining
- astar : A* Search
- cnf   : CNF Generator

## Ví dụ
python main.py Inputs/input-01.txt fc Outputs/output-01.txt
